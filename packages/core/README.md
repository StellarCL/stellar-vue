# @stellar-vue-ui/core

Production-ready Vue 3 components built on [Radix Vue](https://www.radix-vue.com/) with Tailwind CSS.

## Installation

```bash
npm install @stellar-vue-ui/core
```

## Usage

```vue
<script setup>
import { Button, Input, Dialog } from '@stellar-vue-ui/core'
</script>

<template>
  <Button variant="default">Click me</Button>
</template>
```

## Features

- 56 component families covering forms, data display, navigation, feedback, and layout
- 18 composables for common UI patterns
- Built on Radix Vue for accessible, unstyled primitives
- Styled with Tailwind CSS v4 and class-variance-authority
- Full TypeScript support
- Tree-shakeable ESM and CJS builds
