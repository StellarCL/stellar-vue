# @stellar-vue-ui/cli

CLI tool for scaffolding and managing Stellar Vue UI components in your project.

## Installation

```bash
npm install -g @stellar-vue-ui/cli
# or use directly with npx
npx stellar-ui init
```

## Commands

- `stellar-ui init` — Initialize Stellar Vue UI in your project
- `stellar-ui add <component>` — Add a component to your project
- `stellar-ui update` — Update components to the latest version
- `stellar-ui remove <component>` — Remove a component
- `stellar-ui list` — List available components
- `stellar-ui theme` — Manage themes
- `stellar-ui info` — Show project info
- `stellar-ui deps` — Check dependencies
- `stellar-ui audit` — Audit your setup
